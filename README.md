# Documentation

## Development

Install **requirements**
- Ionic cli `npm i -g @ionic/cli`
- Node modules `npm i` 

Create `.env`

```dotenv
VITE_API_URL=http://localhost:5000 # https://github.com/withdule/backend - Publicly avaible at https://dule-api.camarm.fr 
```

### Scripts
Start in dev mode
```shell
npm run dev
```

Build
```shell
npm run build
```

Start in production mode
```shell
ionic serve
```
